# opioid_project

<h5>Class: STOR 320</h5> <br>

<p>This repository contains code for my STOR 320 semester long group project. Specifically it contains my portion of the coede. The project sets out to explore how different factors effect opioid Deaths. We are using a Kaggle data set found 
	<a href="https://www.kaggle.com/ryanandreweckberg/opioid-crisis-by-interpersonal-relationships">here</a>. We are currently in
the exploratory data analysis portion of the assignment. all of the exploratory code so far is mine and is not final, i.e. it won't be what I turn in. It explores the relationship between population density and number opioid deaths by county, and the relationship between the porportion of people born in the us and opioid deaths. At the end of the markdown file I am attempting to build a leaflet map of opioid deaths per 100,000.</p>
